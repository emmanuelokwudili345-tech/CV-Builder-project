A fast, beautiful, and responsive CV builder that lets users create professional CVs in minutes — no sign up required.

🔗 **Live Demo:** [clarix-project.netlify.app](https://clarix-project.netlify.app)

---

---

## Features

-  **Live Preview** — See your CV update in real time as you type
-  **3 Professional Templates** — Classic, Modern, and Minimal
-  **Font Picker** — Choose heading and body fonts from 12 options
-  **Accent Color Picker** — Personalize your CV color
-  **Fully Responsive** — Works beautifully on mobile, tablet, and desktop
-  **Auto Save** — Your progress is saved automatically to localStorage
-  **Photo Upload** — Add a profile photo to your CV
-  **Completeness Score** — Track how complete your CV is
-  **Section Tips** — Helpful guidance for each section
-  **PDF Download** — Download your CV as a PDF
-  **Toast Notifications** — Clean, non-intrusive user feedback
-  **Splash Screen** — Smooth loading animation on startup

---

## Built With

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, Flexbox, Grid, responsive design
- **Vanilla JavaScript** — No frameworks, no dependencies
- **Google Fonts** — DM Serif Display, DM Sans, and more
- **html2pdf.js** — PDF generation
